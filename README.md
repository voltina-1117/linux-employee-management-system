 Linux 员工薪资管理系统
基于C语言+MySQL开发的企业级员工信息管理控制台程序，支持三级权限控制与数据安全校验。

技术栈
- 开发语言：C语言
- 数据库：MySQL（员工表/部门表/薪资表）
- 运行环境：Linux
- 核心特性：三级权限体系、数据增删改查、薪资统计、操作审计

 核心功能
1.  三级权限管理：
    - 老板：全功能管理（员工增删改查、部门管理、薪资统计、数据库初始化）
    - 主管：本部门员工信息管理、薪资查询
    - 员工：个人信息与薪资明细查看
2.  员工信息管理：支持员工信息增删改查、按部门/工号检索
3.  薪资管理：月度薪资明细查询、自动薪资统计、历史薪资记录
4.  数据安全：SQL预处理防注入、事务处理保证数据一致性、登录错误次数限制

编译与运行
bash
 编译命令
gcc employee_manage.c -o employee_manage -lmysqlclient

 运行程序
./employee_manage
